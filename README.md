# Novvera

开源双端轻小说阅读器。

支持 Windows、macOS、Linux、Android、iOS。  
基于 [Venera](https://github.com/venera-app/venera) 改造。

**下载最新版：** https://github.com/YeXuanHs/novvera-app/releases/latest

## 能做什么

- 在线浏览、搜索、阅读轻小说（含插图）
- 内置三个书源，开箱即用
- 本机收藏与阅读历史
- 导出 EPUB：自选文件夹，按「书名 / 卷名 / 章节」保存
- 应用内检查更新并下载对应平台安装包

## 书源

| 书源 | 数据来源 |
|------|----------|
| **轻小说文库** | 官方 API；仅「推荐书籍」走网页，需要账号会话 |
| **哔哩轻小说** | 网页爬取 |
| **幻梦轻小说** | 网页爬取 |

部分站点可能触发人机验证或访问限制，应用内会提示处理。

## 说明

收藏与进度只保存在本机，不依赖云端账号。  
本项目为开源爱好者作品，请支持正版轻小说。
